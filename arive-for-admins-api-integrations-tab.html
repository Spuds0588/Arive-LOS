ARIVE for Admins | API Integrations Tab

Author: Corey Burns | Updated: October 2025

Overview

In this tutorial for administrators of the ARIVE Loan Origination System (LOS), we explore the functionalities of the API Integrations tab located within the company's Admin Controls. This guide provides a detailed walkthrough of how to navigate and manage third-party service integrations, a crucial aspect for customizing and streamlining the loan origination workflow. Understanding these settings allows administrators to connect various tools and services directly into their ARIVE environment, enhancing overall system capability and efficiency.

Key Takeaways

  • The API Integrations tab is located within the Admin Controls section of the ARIVE platform, accessible to administrators.
  • This section allows system administrators to manage connections with essential third-party services like credit bureaus and payment processors.
  • Properly configuring API integrations is key to customizing workflows and automating tasks within the ARIVE LOS.
  • From this tab, administrators can enable, disable, and configure settings for various integrated services to enhance the platform's functionality.

    Frequently Asked Questions

    Q: What is the purpose of the API Integrations tab in the ARIVE LOS?
    A: The API Integrations tab allows administrators to connect, manage, and configure third-party applications and services that work with the ARIVE Loan Origination System, enhancing its functionality and automating processes like credit checks and payment processing.

    Q: Where do I find the API Integrations settings in ARIVE?
    A: The API Integrations settings are located in the main Admin Controls section of the ARIVE platform, which is accessible to users with administrative privileges.
