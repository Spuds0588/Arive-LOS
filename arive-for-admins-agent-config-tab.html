ARIVE for Admins | Agent Config Tab

Author: Corey Burns | Updated: October 2025

Overview

This video is a tutorial for administrators of the ARIVE Loan Origination System (LOS). It specifically covers the functions and settings available within the Agent Config Tab in the Arive Admin Controls, guiding admins on how to properly configure agent-level settings.

Key Takeaways

  • Accessing the Agent Config Tab is done through the main Admin Controls.
  • Admins can set and manage individual agent configurations.
  • Proper setup of the Agent Config Tab is crucial for workflow efficiency.
  • The tutorial focuses on an administrative perspective of the ARIVE platform.

    Frequently Asked Questions

    Q: What is the Agent Config Tab in ARIVE?
    A: The Agent Config Tab is a section within the ARIVE Admin Controls where administrators can manage and configure settings for individual agents on the platform.

    Q: Who is this ARIVE tutorial for?
    A: This tutorial is designed for system administrators who are responsible for managing the ARIVE Loan Origination System (LOS) for their company.
