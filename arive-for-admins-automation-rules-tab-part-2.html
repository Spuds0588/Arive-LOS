ARIVE for Admins | Automation Rules Tab (Part 2)

Author: Corey Burns | Updated: October 2025

Overview

This guide continues the exploration of the Automation Tab within the ARIVE Loan Origination System (LOS) for administrators. The primary focus is on Task Templates, detailing how to create them and effectively implement them to add automated tasks. This tutorial provides administrators with the knowledge to streamline workflows and enhance efficiency by leveraging the full power of ARIVE's automation capabilities.

Key Takeaways

  • Learn to navigate the Automation Rules tab within ARIVE's admin controls.
  • Understand the process of creating and managing Task Templates.
  • Discover how to apply Task Templates to build effective automation rules.
  • Streamline your loan origination workflow by automating task creation.

    Frequently Asked Questions

    Q: What is the primary focus of Part 2 of the Automation Rules Tab tutorial?
    A: This tutorial focuses specifically on creating Task Templates and then using those templates to add automated tasks within the ARIVE LOS automation rules.

    Q: Who is this ARIVE tutorial intended for?
    A: This tutorial is designed for system administrators of the ARIVE Loan Origination System who are responsible for setting up and managing company-wide settings and automation.

    Q: What are Task Templates used for in ARIVE?
    A: Task Templates in ARIVE allow administrators to pre-define sets of tasks that can be automatically added to a loan file when certain conditions are met, ensuring consistency and efficiency in the workflow.
