ARIVE for Admins | Automation Rules Tab (Part 1)

Author: Corey Burns | Updated: October 2025

Overview

This guide provides a detailed walkthrough of the Automation Rules Tab within the ARIVE Loan Origination System (LOS), specifically for administrators. It explores how to access and navigate the automation controls in the company settings. The tutorial serves as the first part of a series dedicated to helping admins leverage ARIVE's powerful automation features to create efficient and streamlined mortgage origination workflows.

Key Takeaways

  • Learn how to locate and access the Automation Rules Tab within the ARIVE admin settings.
  • Understand the fundamental purpose of automation rules in the ARIVE Loan Origination System.
  • Get an introduction to the interface and primary functions available in the Automation Tab.
  • Discover how automation can streamline administrative tasks and improve workflow efficiency.
