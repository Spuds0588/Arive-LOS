Author: Corey Burns | Updated: October 2025

Overview

This video covers the release notes for Arive 8.0.0, the latest update to the all-in-one mortgage origination platform. It highlights the introduction of a new 'Reactions' feature, aimed at improving communication and feedback within the loan origination process for mortgage brokers.

Key Takeaways

  • Arive's 8.0.0 update introduces a new feature called 'Reactions'.
  • This release focuses on enhancing user interaction and communication on the platform.
  • Arive is a comprehensive software platform for mortgage brokers, combining LOS, POS, and a pricing engine.
  • The platform is designed to streamline the entire mortgage process from application to closing.
